# 📋 구글 폼 임베드 방법 (Step-by-Step)

'창업하는 사이다' 페이지에 직접 만드신 구글 폼을 넣는 방법입니다. 아래 순서대로 진행해 주세요!

---

### 1단계: 구글 폼에서 코드 복사하기
1. 제작하신 **구글 폼 편집 화면**으로 들어갑니다.
2. 우측 상단의 **[보내기]** 버튼을 클릭합니다.
3. 상단 탭에서 세 번째에 있는 **`<>` (HTML 삽입)** 아이콘을 선택합니다.
4. 하단의 **[복사]** 버튼을 눌러 공유 코드를 복사합니다.
   - *팁: 너비(Width)는 100%로 수정하면 모바일에서도 예쁘게 나옵니다.*

### 2단계: `index.html` 파일 수정하기
1. `index.html` 파일을 엽니다.
2. `id="apply"` 섹션 안에 있는 `<form>` 태그 부분을 찾습니다. (파일 하단부)
3. 복사한 **iframe 코드**를 해당 위치에 붙여넣습니다.

### 3단계: 코드 적용 예시
기존의 `<form>...</form>` 부분을 지우고 아래와 같이 코드를 넣으시면 됩니다.

```html
<!-- 변경 전 (기존 코드) -->
<form onsubmit="...">
    ...기존 입력창들...
</form>

<!-- 변경 후 (구글 폼 적용) -->
<iframe 
    src="https://docs.google.com/forms/d/e/YOUR_FORM_ID/viewform?embedded=true" 
    width="100%" 
    height="800" 
    frameborder="0" 
    marginheight="0" 
    marginwidth="0">
    로드 중…
</iframe>
```

---

### 💡 전문가의 팁
- **높이 조절**: 폼이 길다면 `height="800"` 부분을 `1000`이나 `1200`으로 늘려보세요.
- **배경 조화**: 구글 폼 자체 설정에서 배경색을 투명하게 하거나, 웹사이트와 비슷한 녹색 계열로 맞추면 훨씬 전문적으로 보입니다.
